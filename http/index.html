Your new blank view awaits!

Step 1 Generate an SSH key if you haven’t already, by running this in your Terminal:

ssh-keygen -t rsa

Step 2 Add your SSH key to your account if you haven’t already, by running this in your browser’s JavaScript console:

$.ajax({url: "/api/<username>/sshkeys", type: "POST", data: {key: "paste ssh key here"}})

Step 3 SSH in!

ssh [box]@[server]
